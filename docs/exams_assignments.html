Coursework and Grading

  • There are no exams or defined homework.
  • Students form small teams (approximately 5 students) and develop a project. A project review at the end of the semester determines completion of requirements and grade.
  • Students must submit weekly status reports and have periodic reviews. Failing to submit status reports or schedule interim reviews results in point deductions.

Project Sprint 0

  • Each student independently completes sprint 0 and submits a completion report. We will provide the details for submitting the report on CourseWorks.
  • There are two mandatory and one optional subtasks for sprint 0.
    1. Mandatory: clone the starter database repository and follow the instructions in the README.md.
    2. Mandatory: clone the starter microservice repository and follow the instructions in the README.md.
    3. Optional: I am working on the sample web browser UI application. I will release later today.
